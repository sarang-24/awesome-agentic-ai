# Contributing to Awesome Agentic AI 🤝

We love contributions! By participating in this project, you help make it a better resource for the entire AI developer community.

Here are a few guidelines to follow when contributing:

## How to Contribute

1. **Check the existing list**: Make sure the framework, tool, or resource isn't already listed.
2. **Fork the repository**: Click the "Fork" button at the top right of this page.
3. **Clone your fork**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/awesome-agentic-ai.git
   ```
4. **Create a branch**:
   ```bash
   git checkout -b add-my-awesome-resource
   ```
5. **Add the resource**: Edit `README.md` and insert your resource into the appropriate category in alphabetical order. Follow this format:
   ```markdown
   * **[Resource Name](URL)** - A short, clear description of what the resource does (keep it objective and concise).
   ```
6. **Commit and Push**:
   ```bash
   git add README.md
   git commit -m "add: [Resource Name] to [Category]"
   git push origin add-my-awesome-resource
   ```
7. **Submit a Pull Request (PR)**: Go to the original repository and click "Compare & pull request". Provide a short description of the resource you are adding.

## Quality Guidelines

- **Active Projects only**: Tools and libraries should have active development (avoid listing abandoned repositories).
- **No duplicates**: Do not add the same resource in multiple categories.
- **Formatting**: Make sure to check that the links are working and formatted correctly.

Thank you for your contribution! You are helping build the future of Agentic AI.
